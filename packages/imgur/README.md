# Imgur
A simple imgur browser client

![Imgur](https://github.com/rgbm21/Imgur/blob/master/imgur.gif)

## Getting Started

```
    > git clone https://github.com/rgbm21/Imgur.git
    > cd Imgur
    > npm install
    > gulp
```
