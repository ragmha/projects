# Chat
Simple chat app with React, Alt and Firebase

![Chat](https://github.com/rgbm21/Chat/blob/master/chat.gif)

## Getting Started

```
    > git clone https://github.com/rgbm21/Chat.git
    > cd Chat
    > npm install
    > npm start
```
