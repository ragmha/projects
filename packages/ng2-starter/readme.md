# 🦄 ng2 starter

### NOTE ✋ ⚠️ `This is a simple starter to experiment and test Angular components`

## Getting Started 🚀

* ```$ git clone https://github.com/ragmha/ng2-starter```

* ```$ yarn```

* ```$ yarn start -s```
